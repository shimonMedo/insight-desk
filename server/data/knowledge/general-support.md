# General Support

InsightDesk is designed around one operational loop:

1. The user asks for help in the support chat.
2. The system answers using the internal knowledge base.
3. The user marks the answer as helpful or not helpful.
4. If the answer did not help, the issue is escalated into a support ticket.
5. Internal teams review repeated failed interactions in the insights dashboard.

Escalation should be used when the answer is incomplete, the user still cannot complete the task, or the workflow is unclear.
