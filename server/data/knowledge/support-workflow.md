# Support Workflow

The support assistant is meant to help users finish inventory tasks without leaving the product flow.

The expected workflow is:

- the user asks a question from the support page
- the assistant answers using product knowledge
- if the answer does not unblock the task, the user escalates the issue
- escalated issues are stored for review
- only repeated friction or clear product gaps should appear in the insights dashboard

Not every escalation is a product bug. Some cases should stay as support follow-up unless the issue is recurring or clearly points to a product gap.
