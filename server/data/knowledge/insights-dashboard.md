# Insights Dashboard

The insights dashboard is the internal review area for failed support interactions.

Internal users can use it to:

- review unresolved support tickets
- identify repeated user complaints
- inspect the AI answer that failed
- track ticket status from new to reviewed to fixed
- decide whether the next improvement is a UI fix, workflow fix, or support-content fix

The main goal is to turn support failures into clear product, workflow, or UI improvements.
