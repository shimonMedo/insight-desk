# Inventory Updates

If an inventory update fails, the user should review the latest activity for that medication and confirm that the quantity and action were correct.

Common recovery steps:

- refresh the inventory board
- retry the last update action
- confirm that the correct quantity was entered
- check whether a previous action is still pending
- escalate the issue if the displayed quantity still does not match the expected stock level

If the inventory state still looks wrong after retrying, the issue should be escalated for human review.
